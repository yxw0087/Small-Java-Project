Mailing List and Support

A mailing list is provided for all users of Mckoi SQL Database.

Mckoi SQL Database Mailing list

  To subscribe, send a blank message to: mckoidb-subscribe@mckoi.com
  To unsubscribe, send a blank message to: mckoidb-unsubscribe@mckoi.com
  List Archives (powered by Mckoi SQL): Browse Archives / Search Archives

All general questions / comments and bug reports should be sent to the list. If you would like to contact us privately you can email toby@mckoi.com.

Old list archives can be accessed here: http://mckoi.com/database/maillist/threads.html
